📌 Web Scraping Repository

This repository contains a collection of web scraping scripts and utilities built using Python. It focuses on extracting structured data from various websites using libraries such as BeautifulSoup and Requests.

The project demonstrates techniques for handling static and dynamic web pages, including pagination, data parsing, and automation of browser interactions. Extracted data is cleaned, processed, and stored in formats like CSV or JSON for further analysis.

Key features include:

Scraping data from multiple pages and categories
Parsing HTML elements with BeautifulSoup
Data storage and preprocessing using Pandas
Error handling and efficient request management
